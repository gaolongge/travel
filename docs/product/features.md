# 产品需求文档：旅途笔记 - 个人旅行博客

## 产品概述

旅途笔记是一个以旅行游记为主题的个人博客网站，采用文艺清新的设计风格。网站旨在为博主提供一个记录和分享旅行经历的平台，通过优美的文字和图片，展现旅途中的美好瞬间与深刻感悟。

**核心价值**：
- 📝 **记录旅程**：便捷地发布和管理游记文章
- 🏷️ **标签分类**：通过目的地、旅行类型等标签快速筛选文章
- 🎨 **沉浸阅读**：支持 Markdown 渲染，提供舒适的阅读体验
- 🌸 **文艺美学**：清新淡雅的视觉设计，营造诗意氛围

---

## 核心功能

### 1. 文章列表（首页）
- 瀑布流卡片式布局展示文章
- 每张卡片显示：封面图、标题、摘要、发布日期、标签
- 支持按标签筛选文章
- 响应式设计，适配各种屏幕尺寸
- 骨架屏加载状态

### 2. 文章详情页
- Markdown 内容渲染（支持代码高亮、图片、表格等）
- 文章元信息：作者、发布时间、阅读时长
- 标签展示
- 上一篇/下一篇导航
- 返回列表按钮

### 3. 标签分类
- 侧边栏标签云展示
- 点击标签筛选相关文章
- 显示每个标签下的文章数量

### 4. 关于页面
- 博主个人信息展示
- 头像、昵称、个人简介
- 社交媒体链接
- 旅行足迹统计

## 页面结构

### 页面路由
| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 首页（文章列表） | 展示所有文章，支持标签筛选 |
| `/article/:id` | 文章详情 | 展示文章完整内容 |
| `/about` | 关于页面 | 博主个人信息 |
| `*` | 404页面 | 页面不存在提示 |

## 数据模型

### Article（文章）
```typescript
interface Article {
  id: string;
  title: string;           // 标题
  slug: string;            // URL 友好的标识
  excerpt: string;         // 摘要（150字以内）
  content: string;         // Markdown 内容
  coverImage: string;      // 封面图片 URL
  tags: Tag[];             // 关联标签
  author: Author;          // 作者
  readTime: number;        // 阅读时长（分钟）
  createdAt: Date;         // 创建时间
  updatedAt: Date;         // 更新时间
}
```

### Tag（标签）
```typescript
interface Tag {
  id: string;
  name: string;            // 标签名称
  slug: string;            // URL 友好的标识
  color: string;           // 标签颜色（可选）
  articleCount: number;    // 文章数量
}
```

### Author（作者）
```typescript
interface Author {
  id: string;
  name: string;            // 昵称
  avatar: string;          // 头像 URL
  bio: string;             // 个人简介
  location: string;        // 所在地
  social: {                // 社交链接
    weibo?: string;
    wechat?: string;
    instagram?: string;
    email?: string;
  };
}
```

## 视觉设计

### 设计风格
- **文艺清新**：温暖柔和的色调，营造诗意氛围
- **简约舒适**：注重阅读体验，避免视觉干扰
- **卡片设计**：文章以卡片形式展示，层次分明

### 配色方案
- **主色调**：温暖的橙色/琥珀色（象征旅途中的阳光与温暖）
- **辅助色**：米白、浅灰、淡褐
- **强调色**：深棕色（用于标题、重点文字）

### 字体
- **标题**：衬线字体（Noto Serif SC），典雅大气
- **正文**：无衬线字体（Noto Sans SC），清晰易读
